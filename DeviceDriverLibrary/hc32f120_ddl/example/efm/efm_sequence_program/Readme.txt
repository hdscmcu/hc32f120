平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例为flash 连续编程，全擦除。

使用步骤
================================================================================
1）打开工程，options->Linker->config，更改linker configuration file，选择
   hc32f120_ram.icf  (mcu\EWARM\config)；
2）options->Debugger->Download，取消勾选“use flash loader(s)”；
3）重新编译；
4）启动IDE的下载和调试功能；
5）在main函数中EFM_SequenceProgram处（117行），EFM_ChipErase（122）行打断点；
6）查看memory地址（0X00001400），memory地址（0X00001600）；
7）运行到第一个断点处，可看到memory窗口全FF，运行到第二个断点处，可看到memory窗口写入数据；
8）单步运行，可看到该地址数据被擦除。

说明：步骤1）2）适用于IAR工程，KEIL工程参考设置https://blog.csdn.net/wjs1033/article/details/51745494
