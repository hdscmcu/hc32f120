平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例展示TimerB模块输出比较功能。

说明：
本样例设置TimerB为锯齿波、向上计数模式，TimerB单元1和2周期计数时间为1S，TimerB单
元1和2比较计数值为周期计数值一半，TimerB单元1和2使能同步计数功能，使能单元2，产
生相位相同的PWM波形。

使用步骤
================================================================================
1）P146(TIMB_1_PWM1)、P16(TIMB_2_PWM1)引脚与示波器相连
2）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）观察示波器，P146、P16引脚通道波形极性相同，占空比50%，周期为2S。
