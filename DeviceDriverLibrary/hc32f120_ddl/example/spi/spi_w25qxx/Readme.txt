功能描述
================================================================================
本例程实现了SPI模块对 W25Q64 SPI Flash 的读写操作。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
STK_HC32F120_LQFP44_080_V11

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的调试功能，断点查看程序执行情况；


================================================================================
注意
================================================================================
本例程所用的对 W25Q64 SPI Flash 写数据的函数，会先将目标地址的扇区全部擦除，然后
再写入，也就是在写入之前不会对扇区进行备份。
