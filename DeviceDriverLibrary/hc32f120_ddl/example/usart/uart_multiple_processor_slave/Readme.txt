平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。


功能描述
================================================================================
本样例主要展示USART外设配置为UART多处理器模式时收发数据。
说明：
1、UART master间隔发送奇/偶数据；
2、UART slave接收数据后，发送回UART master；
3、UART master对比接收数据与发送数据，若一致，数据为偶数，则点亮RGB绿色LED灯，数
   据为奇数，则熄灭RGB绿色LED灯；若不一致，则点亮RGB红色LED灯

使用步骤
================================================================================
1）UART master板J12_P00与J9_TOOL0相连：
2）UART master/slave板参考如下连接：
   UART master板     UART slave板
   UART2_RXD  <-----> UART2_TXD
   UART2_TXD  <-----> UART2_RXD
   UART2_GND  <-----> UART2_GND
3）通过USB数据线，连接PC和板子J1
4）打开串口助手软件，选择UART master板虚拟端口，配置端口如下参数
     波特率：115200
     数据位：8
     校验位：None
     停止位：1
5）打开工程uart_multiple_processor_master并重新编译，下载代码UART master板，并且全速运行。
6）打开工程uart_multiple_processor_slave并重新编译，下载代码UART slave板，并且全速运行
7）按UART master板SW2按键
8）观察UART master板RGB LED灯，绿色LED灯点亮
9）观察串口助手提示信息，接收和发送数据一致，且为偶数
10）按UART master板SW2按键
11）观察UART master板RGB LED灯，绿色LED灯熄灭
12）观察串口助手提示信息，接收和发送数据一致，且为奇数
13）重复步骤7）~12），观察UART master板RGB LED灯状态
