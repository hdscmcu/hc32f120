功能描述
================================================================================
本样例主要展示USART外设配置为UART半双工模式时中断方式收发数据。
说明：
1、UART master单元发送数据给UART slave单元；
2、UART slave单元接收数据后，将数据发送给UART master单元；
3、比较UART master单元发送和接受数据，若一致，数据为偶数，则点亮RGB绿色LED灯，数
   据为奇数，则熄灭RGB绿色LED灯；若不一致，则点亮RGB红色LED灯

测试环境
================================================================================
测试用板:
---------------------
STK_HC32F120_LQFP44_080_V11

辅助工具:
---------------------
无

辅助软件:
---------------------
串口助手

================================================================================
使用步骤
================================================================================
1）需要两个测试板（master板和slave板），按照如下方式连接
   master板UART2_TXD与slave板UART2_TXD相连，slave板引脚J12-P00与J9-TOOL0
2）通过USB数据线，连接PC和板子J1。
3）打开串口助手软件，连接master板虚拟端口，配置端口如下参数。
     波特率：115200
     数据位：8
     校验位：None
     停止位：1
4）打开工程uart_halfduplex_int_master，重新编译，下载代码至master板，并且全速运行。
5）打开工程uart_halfduplex_int_slave，重新编译，下载代码至slave板，并且全速运行。
6）按master板SW2按键
7）观察master板RGB LED灯，绿色LED灯点亮
8）观察串口助手提示信息，接收和发送数据一致，且为偶数
9）按SW2按键
10）观察master板RGB LED灯，绿色LED灯熄灭
11）观察串口助手提示信息，接收和发送数据一致，且为奇数
12）重复步骤6）~11），观察RGB LED灯状态
